
✅ 운영체제란?
운영체제(Operating System, OS)는 컴퓨터 하드웨어와 소프트웨어를 관리하고 사용자와 컴퓨터 간의 인터페이스를 제공하는 시스템 소프트웨어입니다. 주요 역할은 다음과 같습니다:
- 자원 관리: CPU, 메모리, 저장장치 등
- 프로세스 관리: 프로그램 실행 단위를 관리
- 파일 시스템 관리
- 사용자 인터페이스 제공 (예: CLI, GUI)

✅ 컴퓨터 하드웨어 구조
컴퓨터는 일반적으로 다음과 같은 구조를 가집니다:
1. 중앙처리장치(CPU): 연산 및 제어 담당
2. 기억장치: RAM(휘발성), HDD/SSD(비휘발성)
3. 입출력장치(I/O): 키보드, 마우스, 모니터 등
4. 버스(Bus): 장치 간 데이터 이동 통로

✅ CPU 이해
CPU(Central Processing Unit)는 컴퓨터의 두뇌 역할을 하며, 다음과 같은 구성요소로 이루어져 있습니다:
- 연산 논리 장치(ALU): 산술 및 논리 연산 수행
- 제어 장치(Control Unit): 명령어 해석 및 실행 제어
- 레지스터: 임시 데이터 저장소

✅ 프로그램 실행 과정
프로그램이 실행되는 일반적인 과정은 다음과 같습니다:
1. 디스크에서 메모리로 로딩
2. CPU가 명령어 하나씩 읽음(Fetch)
3. 명령어 해석(Decode)
4. 명령어 실행(Execute)
5. 결과 저장 및 다음 명령어 반복

✅ 인터럽트 (Interrupt)
인터럽트는 현재 실행 중인 작업을 중단하고 긴급한 작업을 먼저 처리하는 메커니즘입니다. 종류는 다음과 같습니다:
- 하드웨어 인터럽트: 키보드 입력, 타이머 등
- 소프트웨어 인터럽트: 예외 상황, 시스템 호출 등

✅ 프로세스의 이해
프로세스(Process)는 실행 중인 프로그램을 의미합니다. 주요 특징:
- 고유한 메모리 공간(코드, 데이터, 스택 등)
- 운영체제가 관리하는 스케줄링 대상
- 여러 프로세스는 동시 실행 가능 (멀티태스킹)

프로세스는 생성 → 준비 → 실행 → 종료의 생명 주기를 가집니다.
